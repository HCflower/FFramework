# FFramework
FFramework 是一个用于游戏开发的 C# 框架，旨在提高开发效率、简化项目结构，并支持模块化扩展。
本框架适合中小型 Unity 项目，也可根据实际需求进行定制和扩展。

特性
1.模块化设计：各功能模块解耦，便于维护和扩展。
2.事件系统：支持灵活的事件分发与监听。
3.资源管理：统一的资源加载与释放接口。
4.数据驱动：便于配置和热更新。
5.日志系统：方便调试与定位问题。
6.支持扩展：可快速接入第三方库或自定义模块。
